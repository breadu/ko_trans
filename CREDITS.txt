KO Trans is an open-source OCR and translation tool designed for real-time
language assistance. This software incorporates several third-party
open-source libraries and models. We acknowledge and thank the original
authors for their contributions to the AI and development community.

--------------------------------------------------------------------------------
1. Main OCR Engine: PaddleOCR
--------------------------------------------------------------------------------
* Project: https://github.com/PaddlePaddle/PaddleOCR
* License: Apache License 2.0
* Copyright (c) 2020 PaddlePaddle Authors.
* Description: Provides high-performance OCR capabilities based on the
  PaddlePaddle deep learning framework.

--------------------------------------------------------------------------------
2. Text Detection Model: CRAFT (Character Region Awareness for Text Detection)
--------------------------------------------------------------------------------
* Project: https://github.com/clovaai/CRAFT-pytorch
* License: MIT License
* Copyright (c) 2019-present NAVER Corp.
* Authors: Youngmin Baek, Bado Lee, Dongyoon Han, Sangdoo Yun, Hwalsuk Lee.
* Description: Used for precise localization of text regions within images.
